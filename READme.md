# Simple Digital Clock
A responsive digital clock, which has Hours, Minutes and Secounds. The clock is built with HTML, CSS and JavaScript.
The clock displays the current time and updates every secound in real time.

## Features 
-Live time updates every second 
-Responsive design 
-Simple and clean interface 
-Beginner - friendly project 

## Technologies Used 
-HTML5
-CSS3
-JavaScript
